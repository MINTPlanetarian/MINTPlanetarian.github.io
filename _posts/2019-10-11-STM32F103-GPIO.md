## 一、GPIO是什么？能做什么？ ##
GPIO是通用输入、输出端口的简称，简单地说就是STM32可控制的引脚（但需要注意的是一个GPIO可能包含多个引脚，如STM32芯片中每组GPIO就有16个引脚）。
**每个GPIO都有基本的输入和输出功能。**其中，输出功能是指可以通过引脚控制高、低电平（对应数电中的0和1）；输入功能指的是检测外部输入电平。GPIO的结构框图示意如下：
<center>![GPIO结构框图](http://blog.chinaunix.net/attachment/201806/11/31388439_15286973228T9q.png)</center>
一般来说，GPIO的高低电平的转化都会产生一些控制信息，例如共阳极LED在对应GPIO为高电平时为暗，在对应GPIO低电平时为亮。另外，GPIO是所有通信协议依托的物理串口，也即无论是使用I2C通信协议通信还是SW进行通信，实质上都是依靠GPIO发送或接受信息流。

## 二、GPIO的工作模式 ##
GPIO的工作模式，从大体上可分为输出模式和输入模式，以及一个特殊的复用功能模式。顾名思义，输入模式主要管GPIO的输入功能，其包括**模拟输入**、**浮空输入**、**上拉输入**、**下拉输入**四种；输出模式主要包括**推挽输出**、**开漏输出**两种；复用功能既可以输入也可以输出。<p>
接下来将详细说明每种输入、输出模式的作用（背后寄存器的底层机理忽略）。<p>
**模拟输入：**将模拟信号输入到单片机中，而不经过肖特基触发器（将信号转化为0和1）；<p>
**浮空输入：**输出的高低电平不确定，由外设确定（如按键的信号输入就用浮空输入）；<p>
**上、下拉输入：**上拉输入指的是默认电平由上拉决定（默认高电平），下拉输入指的是默认电平由下拉决定（默认低电平）。关于上下拉的具体内涵，可参见：[如何将单片机的引脚配置为上拉输入](https://zhuanlan.zhihu.com/p/59525510)；<p>
**推挽输出：**双MOS管轮流工作，使I/O口输出的只可能是高电平或低电平（具体是高电平还是低电平由CPU控制，由代码实现）；<p>
**开漏输出：**只有N-MOS管工作，I/O口的输出有两种情况：低电平或者由外部电路控制的电平。它具有 **“线与”特性** ，也就是说，若有很多个开漏模式引脚连接到一起时，只有当所有引脚都输出高阻态，才由上拉电阻提供高电平，此高电平的电压为外部上拉电阻所接的电源的电压。若其中一个引脚为低电平，那线路就相当于短路接地，使得整条线路都为低电平。

## 三、GPIO输出的代码实现（库函数版） ##
	/* 此例程是初始化GPIOA的PIN_1引脚，并将GPIOA设置为推挽输出，50MHz */
	*my_gpio.c*
	void GPIO_Config(void)
	{
		/* 定义一个GPIO_InitTypeDef的结构体，GPIO_InitTypeDef是STM32官方的结构体类型，内含GPIO_Pin,GPIO_Mode,GPIO_Speed三个数据 */
		GPIO_InitTypeDef GPIO_InitStructure;
		
		/* 开启GPIO外设时钟 */
		RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA，ENABLE)

		/* 选择要控制的GPIO引脚 */
		GPIO_InitStructure.GPIO_Pin = GPIO_PIN_1;	

		/* 设置引脚模式为通用推挽输出 */
		GPIO_InitStructure.GPIO_Mode = GPIO_Mode_Out_PP;   

		/* 设置引脚速率为50MHz */
		GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz; 

		/* 调用库函数，初始化GPIO */
		GPIO_Init(GPIOA, &GPIO_InitStructure);	
	}

	/* 接下来的例程将展示如何调节GPIO的高低电平 */
	*main.c*
	int main()
	{	
		GPIO_SetBits(GPIOA, GPIO_Pin1);  /* 调至低电平 */

		GPIO_ResetBits(GPIOA, GPIO_Pin1);  /* 调至高电平 */
	}

## 四、GPIO输入初始化的代码实现（库函数版） ##
	/* 此例程是初始化GPIOA的PIN_1引脚，并将GPIOA设置为上拉输入，50MHz */
	*my_gpio.c*
	void KEY_Init(void)
	{
		/* 定义一个GPIO_InitTypeDef的结构体，GPIO_InitTypeDef是STM32官方的结构体类型，内含GPIO_Pin,GPIO_Mode,GPIO_Speed三个数据 */
		GPIO_InitTypeDef GPIO_InitStructure;
		
		/* 开启GPIO外设时钟 */
		RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA，ENABLE)
		
		/* 选择要控制的GPIO引脚 */
		GPIO_InitStructure.GPIO_Pin=GPIO_PIN_1; 
		
		/* 设置引脚模式为下拉输入 */
		GPIO_InitStructure.GPIO_Mode=GPIO_Mode_IPD;
		
		/* 设置引脚速率为50MHz */ 
		GPIO_InitStructure.GPIO_Speed=GPIO_Speed_50MHz; 
		
		/* 调用库函数，初始化GPIO */
		GPIO_Init(GPIOA, &GPIO_InitStructure); 
	}
